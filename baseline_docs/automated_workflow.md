# Automated workflow

1. Trigger + run_id + manifest  
2. Intake classification (vendor, file type, grain, OCR need)  
3. Extraction to raw staging (no cleaning; preserve lineage)  
4. Schema mapping (template-driven; human gate for new formats)  
5. Standardization to canonical facts (parse/normalize/dedupe)  
6. QA + reconciliation + confidence labels  
7. Aggregations (v1–v8)  
8. Benchmarks + scenarios (v9–v10)  
9. Recommendations (v12)  
10. Report composition (workbook + optional PDF/Slides)
